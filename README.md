# Perfect Tuner 🎻

A professional, high-precision web-based instrument tuner designed for excellence. Whether you're a cellist, violinist, or guitarist, Perfect Tuner provides accurate pitch detection and visual feedback to help you stay perfectly in tune.

![Project Status](https://img.shields.io/badge/status-active-brightgreen)
![License](https://img.shields.io/badge/license-MIT-blue)

## ✨ Features

- **High-Precision Pitch Detection**: Uses advanced autocorrelation algorithms for stable and accurate frequency analysis.
- **Visual Gauge & Meter**: Intuitive visual feedback with a high-contrast gauge and needle for precise tuning.
- **Multi-Instrument Support**: Built-in configurations for Cello, Violin, Guitar, and more.
- **Multilingual Support**: Available in English, Russian, and Hebrew.
- **Noise Awareness**: Intelligent background noise detection to help you find the best environment for tuning.
- **Responsive Design**: Works beautifully on desktops, tablets, and mobile devices.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/n4m14v/perfect-tuner.git
   cd perfect-tuner
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

## 🎹 Usage

1. Grant microphone access to the application.
2. Select your instrument from the picker.
3. Play a string. The gauge will show how close you are to the target pitch.
4. Follow the on-screen instructions: "Tighten", "Release", or "Perfect!".
